# Moon Watcher

月有阴晴圆缺，任务从不迟到。

A cron job monitoring dashboard. Tracks scheduled task execution across
multiple repositories, aggregates run logs, and surfaces timing drift
patterns. Runs silently in the background — like moonlight.

## Features
- Multi-repo cron heartbeat aggregation
- Drift detection with tolerance windows
- Hourly status summaries auto-committed

## Why
Scheduled tasks fail silently. Moon Watcher makes silence visible.

Minimal Python. GitHub Actions native. No external dependencies beyond PyGithub.
